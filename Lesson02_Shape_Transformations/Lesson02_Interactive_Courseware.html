📚 认识正方形

🔲 正方形的特征

⬜
  • ✅ 有 4条边
  • ✅ 4条边 一样长
  • ✅ 有 4个角
  • ✅ 4个角都是 直角（90度）

💡 记忆口诀

"正方形，四边同，四个直角站得正！"

📚 认识长方形

▭ 长方形的特征

▭
  • ✅ 有 4条边
  • ✅ 对边相等（上下一样长，左右一样长）
  • ✅ 有 4个角
  • ✅ 4个角都是 直角

🔍 正方形 vs 长方形

相同点：都有4条边、4个直角
不同点：正方形4条边都相等，长方形只有对边相等
秘密：正方形是特殊的长方形！

📚 认识三角形

🔺 三角形的特征

🔺
  • ✅ 有 3条边
  • ✅ 有 3个角
  • ✅ 三个角加起来是 180度
  • ✅ 具有 稳定性（不易变形）

📚 认识圆形

🔵 圆形的特征

🔵
  • ✅ 没有边（边缘是曲线）
  • ✅ 没有角
  • ✅ 从圆心到边缘的距离 处处相等
  • ✅ 可以 滚动

⚠️ 易错点总结

❌ 常见错误

  • 🔴 把长方形和正方形混淆（记住：正方形四边相等）
  • 🔴 数图形时漏数合起来的大图形
  • 🔴 图形分割时，分割线画得不直
  • 🔴 拼组图形时，边不对齐

✅ 成功秘诀

  • 🟢 比较边长：用尺子量或用手指比
  • 🟢 有序计数：从小到大，从左到右
  • 🟢 拼组对齐：相等的边拼在一起
  • 🟢 多动手：折一折、剪一剪、拼一拼
