## 数独练习（4×4）

### 练习1（基础）

```
┌───┬───┬───┬───┐
│ 1 │   │   │ 4 │
├───┼───┼───┼───┤
│   │ 3 │ 2 │   │
├───┼───┼───┼───┤
│   │ 4 │ 1 │   │
├───┼───┼───┼───┤
│ 3 │   │   │ 2 │
└───┴───┴───┴───┘
```

### 练习2（基础）
```
┌───┬───┬───┬───┐
│   │ 2 │ 3 │   │
├───┼───┼───┼───┤
│ 3 │   │   │ 2 │
├───┼───┼───┼───┤
│ 2 │   │   │ 1 │
├───┼───┼───┼───┤
│   │ 1 │ 2 │   │
└───┴───┴───┴───┘
```
