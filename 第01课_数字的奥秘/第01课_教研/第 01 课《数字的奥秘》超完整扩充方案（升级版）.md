**10. 数字拍手游戏 2.0（反向规律）**

plaintext











```
规则：从30倒数到1，遇到5的倍数或含5的数字拍桌子：
30（拍）、29、28、27、26、25（拍）、24...
进阶：拍桌子时说出“5的倍数！”，含5但非倍数时说“含5数字！”
```

**11. 图形找规律（具象化入门）**

plaintext











```
△ □ ○ △ □ ○ __ __ __ （循环规律）
🌸 🍀 🍀 🌸 🍀 🍀 __ __ __ （重复组规律）
解析：用“找朋友”的方式，把重复的图形圈成一组
```

**12. 数感小游戏（生活场景）**

plaintext











```
超市购物：1元、2元、5元硬币各3枚，能凑出多少种不同的金额？
答案：1、2、3（1+2）、4（2+2）、5、6（1+5）、7（2+5）、8（1+2+5）、9（2+2+5）、10（5+5）等（共13种基础组合）
```

#### 原有题目保留（四宫格数独、数字谜语、视觉版找规律）

#### 新增提高题

**13. 等量代换进阶（图文版）**

plaintext











```
已知：🍎 + 🍌 = 9
      🍌 + 🍇 = 8
      🍎 + 🍌 + 🍇 = 15
求：🍎=？ 🍌=？ 🍇=？
解析：先用第三个式子减第一个式子得🍇=6，再依次推导🍌=2，🍎=7
```

**14. 四宫格数独变形（对角线限制）**

plaintext











```
┌───┬───┐
│ 2 │   │
├───┼───┤
│   │ 2 │
└───┴───┘
新增规则：两条对角线也不能重复（1-4）
解题儿歌补充：对角线，也要看，四个数字不碰面
```

**15. 数字排序谜题（逻辑推理）**

plaintext











```
有4个数：8、5、12、7，按以下要求排序：
① 比10小的数在左边
② 双数都在单数右边
③ 最大的单数在最左边
答案：5、7、8、12
```

**16. 图形计数（观察力 + 有序思考）**

plaintext











```
数一数下图有多少个正方形？
（附示意图：由4个小正方形组成的大正方形）
解题方法：先数小的（4个），再数大的（1个），合计5个
延伸：如果是9个小正方形组成的大正方形，总数是？（14个）
```

### 🌟🌟🌟 挑战练习（能力提升阶段）【新增 3 道题】

#### 原有题目保留（六宫格数独、数字金字塔、开放设计题）

#### 新增挑战题

**17. 六宫格数独变形（不规则宫格）**

plaintext











```
┌───┬───┬───┐
│ 2 │   │ 5 │
├───┼───┼───┤
│   │ 4 │   │
├───┼───┼───┤
│ 1 │   │ 6 │
└───┴───┴───┘
新增规则：额外增加2个不规则区域（如左上角4格、右下角4格），区域内数字不重复
```

**18. 数字密码锁（多步推理）**

plaintext











```
一个3位数密码锁，线索：
① 百位比十位大2
② 个位是十位的2倍
③ 三个数的和是14
求：密码是多少？
解析：设十位为x，则百位=x+2，个位=2x → (x+2)+x+2x=14 → 4x=12 → x=3 → 密码536
```

**19. 图形拼合与展开（空间力进阶）**

plaintext











```
一个正方体的展开图如下，数字1对面是几？数字2对面是几？
（展开图：1在中间，2、3在1左侧，4、5在1右侧，6在1下方）
规律：正方体展开图中，“隔一个是对面” → 1对6，2对4，3对5
```

## 三、新增高难度思考题（适配学有余力的学生）

### 思考题 1：数独创新推理

plaintext











```
设计一个“反数独”：每行、每列、每个宫格都必须有重复数字，但重复的数字不能超过2个，且整个数独只有1个重复数字类型（如只有数字3重复）。
要求：
- 四宫格/六宫格均可
- 写出设计思路和验证过程
评价维度：
✅ 符合规则（基础）
✅✅ 重复位置需要推理发现（进阶）
✅✅✅ 能解释“反数独”与普通数独的逻辑差异（高阶）
```

### 思考题 2：数字规律迁移

plaintext











```
观察斐波那契数列：1,1,2,3,5,8,13,21...
回答：
① 第10个数是多少？（55）
② 这个数列中，第几个数开始超过100？（12个，144）
③ 生活中的斐波那契：为什么花瓣数多是3、5、8、13？（引导观察自然，跨科学融合）
拓展：用斐波那契数列设计一个“兔子繁殖”模拟表（1对兔子每月生1对，新生兔子第2个月开始繁殖）
```

### 思考题 3：多维度等量代换

plaintext











```
已知：
① □ + △ + ○ = 18
② □ + □ = △ + △ + △
③ ○ + ○ = □ + □ + □ + □
求：□=？ △=？ ○=？
解析：
由②得：2□=3△ → △=2/3□
由③得：2○=4□ → ○=2□
代入①：□ + 2/3□ + 2□ = 18 → 11/3□=18 → □=54/11？（故意设计非整数，打破“答案必为整数”的思维定式）
延伸：如果要求答案是整数，如何修改条件？（如总和改为22，则□=6，△=4，○=12）
```

### 思考题 4：空间与数字结合

plaintext











```
一个长方体盒子，长、宽、高都是整数，且满足：
① 长+宽+高=12
② 长×宽=15
③ 宽×高=10
求：这个盒子的体积是多少？（长=3，宽=5，高=2，体积=30）
拓展：用积木搭建这个长方体，观察长、宽、高的数字关系
```

## 四、课堂环节扩充（让课堂更饱满）

### 新增环节 1：知识讲解微视频（5 分钟）

- 内容：用动画演示 “斐波那契数列的自然应用”“正方体展开与折叠”
- 形式：1-2 分钟短视频 + 教师 3 分钟讲解，抽象知识具象化

### 新增环节 2：“数学小讲师” 环节（8 分钟）

- 操作：学生分组，每组选 1 道提高题 / 挑战题，派代表讲解解题思路
- 教具：小白板 + 马克笔，学生边画边讲
- 评价：教师用 “讲解星级贴纸” 鼓励（语言清晰、逻辑完整、方法创新）

### 新增环节 3：生活数学实践（7 分钟）

- 场景 1：用计数圆片模拟 “超市找零”（如 10 元买 3 元的东西，怎么找零？）
- 场景 2：用数字积木搭建 “数字金字塔”，验证规律的正确性
- 目标：让数学从 “纸上” 走到 “生活中”

### 新增环节 4：错题复盘与思路分享（5 分钟）

- 操作：收集课堂练习中高频错误的题目，让学生分析 “为什么错”“怎么改”
- 形式：错题展板 + 学生自愿分享，教师总结常见思维误区（如 “数图形漏数大图形”“等量代换忽略整体与部分关系”）
