📚 方法一：凑整法

🔢 什么是凑整法？

利用加法交换律和结合律，把能凑成整十、整百的数先相加。
核心思想：改变运算顺序，让计算更简单！

🌰 例题：27 + 58 + 73

常规算法：27 + 58 = 85，85 + 73 = 158

凑整法：
27 + 73 + 58 = 100 + 58 = 158

💡 思路：27和73是"好朋友"，凑成100，再加58
加法交换律：a + b = b + a
加法结合律：(a + b) + c = a + (b + c)

✏️ 练习1：凑整法

125 + 48 + 75 =

💡 解题思路：
第一题：36和64是好朋友，凑成100
第二题：125和75是好朋友，凑成200

📚 方法二：带符号搬家

🚚 什么是带符号搬家？

在加减混合运算中，数字前面的符号是它的"行李"，搬家时要一起带走。
核心思想：带着符号移动数字位置，凑成整十整百！

🌰 例题：127 - 48 + 73

常规算法：127 - 48 = 79，79 + 73 = 152

带符号搬家：
127 + 73 - 48 = 200 - 48 = 152

💡 思路：把+73搬到127后面，先算200-48
⚠️ 重要提醒：
每个数前面的符号是它的"专属行李"，搬家时必须带着符号一起走！
127+73 中的"+"是73的符号，搬家时要带着！

✏️ 练习2：带符号搬家

用带符号搬家计算：

156 - 78 + 44 =

234 + 67 - 134 =

💡 解题思路：
第一题：把+44搬到156后面，156+44=200，再减78
第二题：把-134搬到234后面，234-134=100，再加67

📚 方法三：基准数法

🎯 什么是基准数法？

当几个数都接近某个整十、整百数时，可以把这个数当作"基准数"，
先算基准数的倍数，再调整差值。
核心思想：化繁为简，先算整再调零！

🌰 例题：98 + 99 + 100 + 101 + 102

观察：这些数都接近100

基准数法：以100为基准数
= 100×5 + (-2-1+0+1+2) = 500 + 0 = 500

💡 思路：5个100是500，差值相互抵消
基准数法公式：
总和 = 基准数 × 个数 + 各数与基准数的差之和

✏️ 练习3：基准数法

用基准数法计算：

97 + 98 + 99 + 100 + 101 =

195 + 196 + 197 + 198 =

💡 解题思路：
第一题：以100为基准，5个数，差值(-3-2-1+0+1)=-5，500-5=495
第二题：以200为基准，4个数，差值(-5-4-3-2)=-14，800-14=786

📚 方法四：等差数列求和

📊 什么是等差数列？

相邻两个数的差都相等的数列叫做等差数列。
例如：1, 2, 3, 4, 5...（公差为1）
例如：2, 4, 6, 8, 10...（公差为2）

等差数列求和公式：
和 = (首项 + 末项) × 项数 ÷ 2
🌰 例题：高斯算法

计算：1 + 2 + 3 + ... + 100

思路：首尾配对，每对和都是101
(1+100) = 101, (2+99) = 101, (3+98) = 101...
共50对，所以：101 × 50 = 5050
🧮 公式推导：
和 = (首项 + 末项) × 项数 ÷ 2
因为：首项+末项 = 第二项+倒数第二项 = ...

✏️ 练习4：等差数列求和

用等差数列求和公式计算：

2 + 4 + 6 + ... + 20 =

5 + 10 + 15 + ... + 50 =

💡 解题思路：
第一题：首项2，末项20，项数10，和=(2+20)×10÷2=110
第二题：首项5，末项50，项数10，和=(5+50)×10÷2=275

📚 方法五：分组计算

🎲 什么是分组计算？

根据数字特征，把算式中的数进行合理分组，
让每组的计算结果相同或简化计算。
核心思想：化繁为简，分组击破！

🌰 例题：20 - 19 + 18 - 17 + ... + 2 - 1

观察：每两个数一组，结果都是1

分组计算：
(20-19) + (18-17) + (16-15) + ... + (2-1)
= 1 + 1 + 1 + ... + 1 (共10个1) = 10
🌰 例题2：1 - 2 + 3 - 4 + 5 - 6 + ... + 99 - 100

分组：(1-2) + (3-4) + (5-6) + ... + (99-100)
= (-1) + (-1) + (-1) + ... + (-1) (共50个-1) = -50

✏️ 练习5：分组计算

用分组法计算：

100 - 99 + 98 - 97 + ... + 2 - 1 =

2 - 4 + 6 - 8 + ... + 98 - 100 =

💡 解题思路：
第一题：每两个数一组，每组差为1，共50组，50×1=50
第二题：每两个数一组，每组差为-2，共25组，25×(-2)=-50

📋 模块二总结：进阶巧算方法

🎯 核心知识点

1. 等差数列求和：和 = (首项 + 末项) × 项数 ÷ 2

2. 分组计算：根据数字特征合理分组，化繁为简

💡 教学要点

• 等差数列求和公式是高斯算法的推广

• 项数 = (末项 - 首项) ÷ 公差 + 1，不要忘记加1

• 分组时要观察数字规律，找到合适的分组方式

• 分组后每组的结果通常相同，便于计算

2. 能正确计算等差数列的项数

⚠️ 易错点总结

常见错误提醒：

❌ 错误1：符号搬家不带符号
正确：127+73-48（+是73的符号）
每个数前面的符号是它的"专属行李"，搬家时必须带着走！
❌ 错误2：基准数法忘记调整差值
正确：总和 = 基准数×个数 + 各数与基准数的差之和
最后一定要加上或减去各数与基准数的差！
❌ 错误3：等差数列项数算错
正确：项数 = (末项-首项)÷公差 + 1
从a到b的连续整数，项数是(b-a+1)，不要忘记加1！
❌ 错误4：分组时数错组数
正确：要数清楚有多少组，避免漏算
建议边分组边标记，确保不遗漏！
