⚡ 第12课 趣味速算

家长协同方案 · 奥数巅峰版（华罗庚金杯难度）

🏆 奥数思维训练

🎯 本课学习目标

通过本课学习，孩子将掌握：

  • 凑整法：利用加法交换律和结合律，将能凑成整十、整百的数先相加
  • 带符号搬家：在加减混合运算中，带着符号移动数字的位置简化计算
  • 基准数法：当多个数都接近某个整十/整百数时，以此数为基准进行巧算
  • 等差数列求和：掌握等差数列求和公式，快速计算连续数之和
  • 分组计算：根据数字特征进行合理分组，化繁为简

📚 核心知识点详解

🔢 方法一：凑整法（好朋友数）

利用加法交换律和结合律，把能凑成整十、整百的数先相加。

例题：27 + 58 + 73
巧算：27 + 73 + 58 = 100 + 58 = 158
思路：27和73是"好朋友"，凑成100
常见好朋友数对：
1+9=10, 2+8=10, 3+7=10, 4+6=10, 5+5=10
11+89=100, 12+88=100, 25+75=100, 36+64=100

🚚 方法二：带符号搬家

在加减混合运算中，数字前面的符号是它的"行李"，搬家时要一起带走。

例题：127 - 48 + 73
巧算：127 + 73 - 48 = 200 - 48 = 152
思路：把+73搬到127后面，先算整百
⚠️ 注意：每个数前面的符号是它的"专属行李"，搬家时必须带着符号一起走！

🎯 方法三：基准数法

当几个数都接近某个整十、整百数时，可以把这个数当作"基准数"。

例题：98 + 99 + 100 + 101 + 102
巧算：以100为基准数
= 100×5 + (-2-1+0+1+2) = 500 + 0 = 500
基准数法公式：
总和 = 基准数 × 个数 + 各数与基准数的差之和

📊 方法四：等差数列求和

连续的自然数或等差数列，可以用公式快速求和。

等差数列求和公式：
和 = (首项 + 末项) × 项数 ÷ 2
例题：1 + 2 + 3 + ... + 100
巧算：(1 + 100) × 100 ÷ 2 = 101 × 50 = 5050
高斯算法：首尾配对，每对和都是101，共50对

🎲 方法五：分组计算

根据数字特征，把算式中的数进行合理分组。

例题：20 - 19 + 18 - 17 + ... + 2 - 1
巧算：每两个数一组：(20-19) + (18-17) + ... + (2-1)
= 1 + 1 + ... + 1 (共10个1) = 10

📝 课堂重点回顾

本节课孩子们学习了以下内容：

  • 认识了5种速算巧算方法：凑整法、带符号搬家、基准数法、等差数列求和、分组计算
  • 掌握了好朋友数的概念，能快速识别能凑成整十整百的数对
  • 学会了带符号搬家的技巧，理解符号是数的"专属行李"
  • 理解了基准数法的适用场景，能选择合适的基准数
  • 记住了等差数列求和公式，能快速计算连续数之和
  • 学会了根据数字特征进行合理分组
💡 教学提示：
速算的核心是观察数字特征，选择合适的方法。不要死记硬背，要理解每种方法的原理和适用场景。

🏠 家庭巩固练习

请家长陪同孩子完成以下活动：

🎮 活动一：好朋友数配对游戏

材料：数字卡片（1-100）

玩法：

  • 随机抽取两张卡片，快速判断是否能凑成整十或整百
  • 如：抽到27和73，孩子要快速说出"好朋友！凑成100！"
  • 限时1分钟，看能找出多少对好朋友数

目标：训练孩子对数字的敏感度，快速识别能凑整的数对

✏️ 活动二：速算小达人挑战

题目：家长出题，孩子用巧算方法解答

示例题目：

  • 36 + 87 + 64 = ? （凑整法）
  • 156 - 78 + 44 = ? （带符号搬家）
  • 97 + 98 + 99 + 100 + 101 = ? （基准数法）
  • 2 + 4 + 6 + ... + 20 = ? （等差数列）
  • 100 - 99 + 98 - 97 + ... + 2 - 1 = ? （分组法）

目标：巩固课堂所学方法，提高计算速度和准确率

🔍 活动三：方法识别训练

玩法：

  • 家长写出一个算式，让孩子判断应该用哪种巧算方法
  • 孩子说出方法名称和理由
  • 再用该方法进行计算

示例：

  • 45 + 78 + 55 → 凑整法（45+55=100）
  • 134 - 67 + 66 → 带符号搬家（134+66=200）
  • 95 + 96 + 97 + 98 + 99 → 基准数法（都接近100）

目标：培养孩子观察数字特征的能力，学会选择合适的方法

⚠️ 常见错误提醒

  • 符号搬家不带符号：记住每个数前面的符号是它的"专属行李"，搬家时必须带着走
  • 凑整后忘记调整：基准数法中，最后一定要加上或减去各数与基准数的差
  • 等差数列项数算错：从a到b的连续整数，项数是(b-a+1)，不要忘记加1
  • 分组时漏数：分组计算时，要数清楚有多少组，避免漏算
  • 方法选择不当：不是所有题目都适合巧算，有时直接计算更简单

👨‍👩‍👧 亲子互动建议

  • 生活中的速算：购物时让孩子计算总价，练习凑整法
  • 速算比赛：家长和孩子比赛，看谁算得又快又准
  • 错题分析：孩子做错时，引导他分析错误原因，而不是直接告诉答案
  • 方法总结：鼓励孩子总结每种方法的适用场景，制作"速算方法卡"
  • 循序渐进：从简单题目开始，逐步增加难度，建立孩子信心
